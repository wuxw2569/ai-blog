👉 示例代码，轻松上手：👇
```python
def generate_test_cases(text):
    processed_text = preprocess_text(text)
    test_cases = []
    for i in range(len(processed_text) - 1):
        test_cases.append(f"Check if '{processed_text[i]}' and '{processed_text[i+1]}' are related.")
    return test_cases
```
