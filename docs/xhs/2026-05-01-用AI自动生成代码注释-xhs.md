🌟 实现步骤：
1️⃣ 代码解析：提取关键信息，如变量、函数、类等。
2️⃣ 上下文理解：结合代码上下文，理解功能与目的。
3️⃣ 注释生成：根据信息与上下文，生成描述性注释。

```python
import ast

def parse_code(code):
    tree = ast.parse(code)

def generate_comment(node):
    if isinstance(node, ast.FunctionDef):
        return f"This function {node.name}..."
    elif isinstance(node, ast.ClassDef):
        return f"This class {node.name}..."
    # ... 其他节点类型

def main():
    code = """
def add(a, b):
    return a + b

class Person:
    def __init__(self, name):
        self.name = name
    """
    tree = parse_code(code)
    for node in ast.walk(tree):
        if node != tree:
            comment = generate_comment(node)
            print(f"{comment}\n{node}")
            print("------")

if __name__ == "__main__":
    main()
```
