### 🤖 AI是怎么做到的？

1. **分析代码**：AI会仔细研究你的代码，提取出关键信息。
2. **注释模板**：根据这些信息，它会自动生成注释模板。
3. **自然语言处理**：然后，AI会用自然语言处理技术，将模板中的信息填满，生成完整的注释。

### 🌈 实现步骤超简单

给大家分享一个Python小例子，看看AI是怎么工作的：

```python
import re

def generate_comment(code):
    pattern = r"def (\w+)\((.*?)\):"
    match = re.search(pattern, code)
    if match:
        function_name = match.group(1)
        params = match.group(2).split(',')
        comment_template = f"""
        # {function_name}({', '.join(params)})
        # 功能描述：
        # 参数：
        # a: 整数
        # b: 整数
        # 返回值：
        """
        comment = comment_template.format(
            function_name=function_name,
            params=', '.join(params)
        )
        return comment
    else:
        return "无法生成注释"

code = "def add(a, b): return a + b"
print(generate_comment(code))
```
