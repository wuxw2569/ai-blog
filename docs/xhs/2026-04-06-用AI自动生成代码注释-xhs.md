### AI如何做到的？

1. **代码分析**：AI会仔细阅读你的代码，理解它的功能和实现方式。
2. **模板匹配**：然后，它会根据预先定义的注释模板，自动填充内容，生成注释。

### 快速上手！

下面是一个简单的Python示例，让你感受一下AI的魔力：

```python
import re

# 预定义的注释模板
templates = {
    'function': "This function {description} and returns {return_type}.",
    'class': "This class {description} and contains the following methods: {methods}."
}

# 代码分析函数
def analyze_code(code):
    functions = re.findall(r'def ([^ ]+)\(.*\):', code)
    classes = re.findall(r'class ([^ ]+):', code)
    return functions, classes

# 生成注释函数
def generate_comments(code):
    functions, classes = analyze_code(code)
    comments = []

    for func in functions:
        description = "calculates the sum of two numbers"  # 这里可以替换为更复杂的逻辑
        return_type = "int"
        comments.append(templates['function'].format(description=description, return_type=return_type))

    for cls in classes:
        description = "represents a user"
        methods = "login(), logout()"
        comments.append(templates['class'].format(description=description, methods=methods))

    return '\n'.join(comments)

# 示例代码
code = """
def add(a, b):
    return a + b

class User:
    def login(self):
        pass

    def logout(self):
        pass
"""

# 生成并打印注释
print(generate_comments(code))
```

### 效果展示

运行上述代码，你将得到以下注释：

```
This function calculates the sum of two numbers and returns int.
This class represents a user and contains the following methods: login(), logout()
```
