🤖 AI自动生成代码注释的原理，其实是通过自然语言处理（NLP）技术和机器学习模型来实现的。简单来说，就是让AI学会理解代码，然后自动生成相应的注释。

📝 下面，我们就来简单看看如何实现这个过程：

1. **准备数据集**：收集一些已注释的代码块，作为训练AI的数据。
2. **训练模型**：用NLP技术处理这些数据，训练出一个能根据代码生成注释的模型。
3. **生成注释**：用训练好的模型，对新的代码块进行注释生成。

🌟 代码示例来啦！👇

```python
import numpy as np
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import MultinomialNB

# 准备数据集
data = [
    ("def add(x, y):", "This function adds two numbers."),
    ("def multiply(x, y):", "This function multiplies two numbers."),
    # ... 更多代码块和注释
]

# 分离代码和注释
texts, labels = zip(*data)

# 转换文本为向量
vectorizer = CountVectorizer()
X = vectorizer.fit_transform(texts)
y = np.array(labels)

# 划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# 训练模型
model = MultinomialNB()
model.fit(X_train, y_train)

# 生成注释
def generate_comment(code):
    vectorized_code = vectorizer.transform([code])
    comment = model.predict(vectorized_code)[0]
    return comment

# 示例
code = "def subtract(x, y):"
print(generate_comment(code))  # 输出：This function subtracts two numbers.
```
